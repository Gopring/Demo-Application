# Demo-Application

This repository contains the source code for the __Demo Application__. This application demonstrates how to use the 1:N streaming and chat features provided by the __Streaming Server(PDN-WebRTC)__. It also includes sample interfaces for API Key management and streaming session testing, allowing users to interact with and explore the platformâ€™s functionality.

## Features

- __1:N Streaming Interface__: Allows users to start a live streaming session or view an ongoing stream.
- __Real-time Chat__: Provides a chat interface for viewers to communicate in real-time during a streaming session.
- __API Key Management__: Includes a demo section where users can test API Key issuance and management.
